# Criação de épicos

## Objetivo

Transformar uma necessidade de negócio em um épico estruturado para Azure DevOps.

## Prompt

```text
Crie um épico para Azure DevOps com linguagem clara e objetiva.

Inclua:
- contexto
- problema ou oportunidade
- objetivo do épico
- escopo de alto nível
- fora de escopo
- riscos
- dependências
- critérios de sucesso
```
