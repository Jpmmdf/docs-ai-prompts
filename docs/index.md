# docs-ai-prompts

Este repositório organiza prompts e instruções reutilizáveis como documentação viva.

## Objetivos

- Padronizar a escrita e evolução de prompts
- Facilitar busca, reuso e governança
- Permitir versionamento via Git
- Publicar uma documentação navegável com MkDocs

## Modelo sugerido para cada prompt

Cada documento pode seguir esta estrutura:

1. Objetivo
2. Quando usar
3. Entradas esperadas
4. Prompt base
5. Regras de saída
6. Exemplos
7. Observações
