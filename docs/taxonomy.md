# Taxonomia de prompts

## Prefixos recomendados

- `prompt-`: prompt operacional reutilizável
- `guide-`: instrução de uso
- `playbook-`: sequência operacional padronizada
- `template-`: estrutura base para criação de novos artefatos

## Domínios

- `linkedin`
- `azure-devops`
- `platform`
- `architecture`
- `research`

## Exemplo de slug

`prompt-linkedin-profile-review`
