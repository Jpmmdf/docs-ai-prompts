# Contribuição

## Regras mínimas

- Um prompt por arquivo quando o conteúdo for independente
- Nome de arquivo curto, descritivo e em kebab-case
- Sempre incluir objetivo e exemplos
- Evitar duplicação; preferir referência cruzada

## Estrutura recomendada

```md
# Título

## Objetivo

## Quando usar

## Entradas esperadas

## Prompt

## Saída esperada

## Exemplos
```
